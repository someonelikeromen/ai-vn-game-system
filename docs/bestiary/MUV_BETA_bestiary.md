## 总览

BETA（Beings of Extra Terrestrial origin which is Adversary of human race）是来自地外的生命体，自 1973 年侵略地球以来，以近乎无法阻挡的姿态蚕食人类文明。已确认种类共 **13 种以上**，按形态与职能分为四大类群。

| 类群 | 种类 |
|------|------|
| 小型种 | 兵士级、斗士级、战车级 |
| 大型种 | 突击级、要击级、要塞级 |
| 光线属种 | 光线级、重光线级、超重光线级 |
| 特殊种 | 母舰级、门级、头脑级、超重脑级 |

> **分类原则**：以外观形态为准，不涉及 BETA 内部生物学联系或亲缘关系。BETA 不存在固有种间特征，未来仍可能出现全新种类。

## 一、小型种

### 🔴 兵士级（Soldier-class）

| 项目 | 数值 |
|------|------|
| 全高 | 2.3 m |
| 全长 | 1.2 m |
| 全宽 | 1.4 m |
| 学名 | *Venarius caput grande* |
| 俗称 | Venator |
| 首次确认 | 1995 年 |
| 行动模式 | 多足步行 |
| 危险等级 | ★★★☆☆（对步兵） |

**外形描述**

兵士级整体呈苍白乳白色与灰紫色相间的体色，表皮布满细密的血管状纹路，质感如同半透明的湿润软组织，全身无硬化甲壳结构。

- **头部**：硕大而圆润，形如倒扣的蘑菇伞盖，颅顶光滑隆起，色泽较身体更为苍白。头部两侧各有一只小而圆的深色眼睛，位置偏低，视野广阔。
- **口部**：位于头颅正下方，开口宽大，常保持半张状态，暴露出一排整齐排列、泛黄橙色的厚实牙齿，给人以持续的压迫感。
- **颈部与上身**：颈部细长，上半身骨架感明显，肋状结构隐约可见于皮下，肌肉组织呈条索状分布，整体轮廓瘦削。
- **前肢**：细长有力，指端弯曲，具有极强的抓握与撕扯能力。
- **下半身**：与上半身形成鲜明对比——腹部至臀部极度膨大，呈球形肿胀，表面可见红色的腔状开口（疑为生物功能性器官），整体色泽偏肉粉，质感柔软。后侧与前身体色和质感一致，无甲壳或硬化结构。

> 总体印象：上半身精瘦、头颅巨大、笑容诡异；下半身臃肿膨胀，全身体表统一柔软——整体呈现出一种令人深感不适的生物错位感。从其外观与出现时间来看，部分专家指出其有可能是参考人类形态而诞生的。

**特征**

- 1995 年首次确认，已知 BETA 中对人类**探测能力最高**的种类
- 平面移动的安静性与速度均远超人类
- 前肢与颚部力量为人类的**数倍至十数倍**，对步兵构成极高威胁

**战术建议**

> 步兵务必避免单独遭遇。战术机可轻松处置，但需警惕其混入战车级群体中的情况。

### 🔴 斗士级（Fighter-class）

| 项目 | 数值 |
|------|------|
| 全高 | 2.5 m |
| 全长 | 1.7 m |
| 全宽 | 1.5 m |
| 学名 | *Agilisis naris prolix* |
| 俗称 | Barrus naris |
| 行动模式 | 二足步行 |
| 危险等级 | ★★★☆☆（对步兵近战） |

**外形描述**

斗士级整体呈苍白色，体表干燥，质感接近粗糙的皮革，无明显血管纹路，与兵士级的湿润半透明体表截然不同。

- **头部**：同样呈倒扣的蘑菇伞盖状，顶部宽阔平展，颅面苍白。头顶背侧散布大量圆形红色肉质瘤突，鲜红醒目，成簇分布，疑为感觉器官或腺体。顶部正面点缀有紫色斑块状色素沉积。
- **口鼻/感觉器**：头颅正前方无明显独立口部，整体面部构造简化；感知功能可能依赖头顶瘤突群完成。
- **象鼻肢**：最显著特征——自头颈部延伸出一条粗壮的象鼻状肌肉附肢，表皮有环状皱褶，柔韧且爆发力极强。**末端分化为数根指状结构，形如手掌**，是实施近战致命攻击的核心武器，可轻松将人类头颅从躯干上扯离。
- **躯干下部突出体**：腹部下段生有一小段粗短的柱状肉质突出体，末端为鲜红色倒钩，钩尖锐利，疑为辅助攻击或固定猎物的器官。
- **躯干**：肌肉紧实，腰背部覆盖一层鳞片状叠层甲质结构，具有一定防护性。腹侧相对柔软，色泽偏白，与整体体色一致。
- **后肢**：粗壮有力，大腿肌肉发达，小腿收细，末端为宽厚蹄状足，适合高速奔跑与急停转向。

> 总体印象：体型紧凑、肌肉感强；象鼻状前肢末端化为手状结构，是其招牌式的近战利器；腹部倒钩突出体增添了一分隐蔽的危险性；全身苍白，外观反差强烈，透出冷峻的捕猎者气质。

**特征**

- 已确认 BETA 中**最小的二足步行种**
- 防御力低，但对人类探测能力与敏捷性极高
- 象鼻状三指前肢可将人类头部直接扯断，是步兵近战的最大威胁
- 步兵标准枪械即可击杀，对战术机无实质威胁

**战术建议**

> 对战术机：低优先级目标，优先清场大型种。  
> 对步兵：**禁止近战**，保持距离以轻武器压制。

### 🔴 战车级（Tank-class）

| 项目 | 数值 |
|------|------|
| 全高 | 2.8 m |
| 全长 | 4.4 m |
| 全宽 | 1.9 m |
| 学名 | *Manderium ungulam crus* |
| 俗称 | Equus pedis |
| 行动模式 | 多足步行（中型） |
| 移动速度 | ≈ 80 km/h |
| 危险等级 | ★★★★☆（数量压制） |

**外形描述**

战车级全身呈深暗红色，体表无皮肤覆盖，直接暴露出肌肉纤维组织，如同一具活体解剖标本，质感粗糙而湿润。

- **头部**：体型相对偏小，圆球状，位于躯干前上方。颅面布满密集的蓝灰色圆形瘤突，成簇排列，与兵士级/斗士级的瘤突形态相似，疑为同类感觉器官。
- **颚部（主武器）**：战车级最具标志性的特征——**宽大颚部直接嵌入躯干正前下方**，并非长于头部。开口极宽，暴露出一排整齐的白色厚实牙齿，齿列坚硬锋利，是咬碎金属与混凝土的主要武器。
- **躯干**：主体宽扁而圆润，肌肉组织层次分明，整体轮廓低矮贴地。躯干前段与颚部一体化，后段向上微微隆起。
- **步行肢**：共有多条细长蜘蛛状步行肢，从躯干两侧对称伸出，肢节纤细但极为强健，末端为圆钝的深紫色蹄足。多足结构使其在各类地形上均能保持高速稳定的行进能力。

> 总体印象：低矮、宽体、多足、血红——战车级的外观如同一只被剥去皮肤的巨型蜘蛛，以那张嵌在腹部的宽阔白牙巨口为中心，整体散发出强烈的原始捕猎者气息。

**特征**

- BETA 群中**数量最多**的中型多足步行种，渗透力与机动力优秀
- 即使在不平整地面仍能以约 80 公里/时持续行进
- 最强武器是**能咬碎金属与混凝土的强韧颚部**，装甲车辆、战术机、步兵均可啃噬
- 个体防御力较低，步兵重武器即可应对；但通常**数十至数百个成群行动**

**战术建议**

> 战车级的核心威胁在于**数量**。单体易于击杀，但大规模群体能在短时间内淹没防线，尽可能避免近距离交战。弹药消耗管理为首要考量。

## 二、大型种

### 🟠 要击级（Grappler-class）

| 项目 | 数值 |
|------|------|
| 全高 | 12 m |
| 全长 | 19 m |
| 全宽 | 28 m |
| 学名 | *Rabidusius bracchium acutas* |
| 俗称 | Medium |
| 行动模式 | 多足步行 |
| 危险等级 | ★★★★★（近战最强） |

**外形描述**

要击级整体呈明显的**双色对比结构**：上半身为乳白至米黄色的柔软肌肉组织，下半身及武装部位为深灰近黑的厚重甲质结构，两者形成强烈视觉对比。

- **头部与颈部**：细长的颈部从躯干顶端直立伸出，颈表可见纵向肌肉条束。头颅形如倒置的骷髅，无明显眼部结构，面部中央为一张横向宽大的裂口式嘴，暴露出整齐的多排白色锯齿状牙列，常保持大幅开张状态。颈部两侧各生有数根细长棘刺状突起，疑为感觉器官。
- **上半身躯干**：宽厚且肌肉感极强，乳白色，与颈部色泽一致；肌肉块组织隆起清晰，具有粗犷的力量感。躯干整体相对紧凑，向下与深色甲壳基座连接。
- **前臂冲角（核心武装）**：最显著特征——两组**巨大的深灰色装甲前臂**从躯干两侧大幅横向展开，每组前臂末端形成宽阔的铲刀状或叶片状冲角。甲质结构多层叠合，表面粗糙厚实，兼具高硬度与高韧性。前臂展开时全宽可达 28 米，是进行大范围横扫与刺击的主要武器。
- **下半身基座**：宽阔低矮的深灰色多甲板结构，覆盖大面积装甲板，整体如一个扁平的重型战车底盘。多条粗壮步行肢从甲板下方伸出，支撑整个庞大身躯。
- **步行肢**：深灰色，粗短有力，末端为扁平宽足，适合支撑巨大体重并快速原地旋转。

> 总体印象：苍白躯干与黑灰重甲的强烈色块对比，骷髅式大口与巨型横展冲角，使要击级在视觉上既有生物的诡异感又有战争机械的压迫感——它就是一台以血肉铸就的近战绞肉机。

**特征**

- 大型种中占比约 **60%** 的主力多足步行种，对人类探测能力较高
- 最大武器是**两对前臂冲角**的打击，前臂攻击范围直径达 **39 米**
- 可进行惊人的原地旋转，被认为是已确认种类中**近距离格斗能力最强**的种类
- 前臂冲角兼具高硬度与高韧性，防弹性能优秀

**战术建议**

> 优先级目标。前臂范围极广，推荐从**侧面或后方**攻击。绝对不要让要击级进入近战距离。

### 🟠 突击级（Destroyer-class）

| 项目 | 数值 |
|------|------|
| 全高 | 16 m |
| 全长 | 18 m |
| 全宽 | 17 m |
| 学名 | *Impetusis arma duras* |
| 俗称 | Ruitare |
| 行动模式 | 多足步行 |
| 移动速度 | ≈ 170 km/h（直线） |
| 危险等级 | ★★★★★（突破冲击） |

**外形描述**

突击级整体给人的第一印象几乎完全由其**正面装甲壳**所主导——巨大的翼状甲板从身体前方向左右大幅展开，遮蔽了几乎所有躯干细节，实际生物本体极为低矮紧凑地隐藏其后。

- **正面装甲壳（核心特征）**：数片巨大的叶/翼形甲板从躯干正面向外展开，形如张开的鳍或盾牌群，完全覆盖正面投影面积。甲板底色为灰绿色，表面布满杂乱的紫色斑纹与**圆形同心圆状斑点**——每一个圆斑均为炮弹命中后装甲再生愈合留下的疤痕组织，中心呈深红橙色，外缘为深紫色晕圈。斑点数量越多，代表该个体经历的战场洗礼越多。甲板边缘锋利，侧面延伸出尖锐棘角。
- **躯干本体**：被装甲壳大幅遮蔽，仅从侧面与正面底部可见。整体呈土黄至棕褐色，表面粗糙，具有鳞裂状纹理。躯干极为低矮紧实，重心压低，是适合高速直线冲锋的流线化结构。
- **头部**：极小，几乎完全没入装甲壳阴影之中。从侧面观察可见正前方有一枚小型圆润突起，表面有单一深色点状感知器，极不显眼——印证了其对人类探测能力为所有已知种中最低的描述。
- **步行肢**：数条粗壮的柱状腿从躯干下方伸出，土黄色，表面分节明显，关节处生有深绿色刀片状角质鳞片，使腿部同样具备一定的伤害能力。
- **背部**：后方躯干裸露区域密布大量深色针刺状或羽片状硬质棘突，形成一道粗糙的背脊防护层。

> 总体印象：突击级本体几乎完全躲藏在由巨大翼状甲板构成的"移动要塞盾"之后；满布再生弹痕圆斑的灰绿装甲宣告了它历经无数炮击而屹立不倒的恐怖生命力；一旦直线加速，它就是一堵以 170 公里时速扑来的活体装甲墙。

**特征**

- 担任 BETA 群**先锋**的大型多足步行种
- **正面装甲壳极为强力**，与要击级前臂硬度相当，面压制下生存率极高
- 装甲壳具有惊人的**再生能力**——表面眼状凹痕均为炮弹命中后再生的痕迹
- 平地直线可以 170 公里/时持续长时间行进，主要攻击手段为**自身冲撞突击战术**
- 回旋与敏捷性显著较低；对人类探测能力为已确认种类中**最低**，步兵携带的反坦克导弹亦可有效攻击

**战术建议**

> 正面拦截代价极高。推荐从**背后攻击**。步兵可利用其低探测能力以反坦克武器伏击。**绝对不要让突击级群突入防线**。

### 🟠 要塞级（Fort-class）

| 项目 | 数值 |
|------|------|
| 全高 | 66 m |
| 全长 | 52 m |
| 全宽 | 37 m |
| 学名 | *Pergrandium tria corpus* |
| 俗称 | Gravis |
| 行动模式 | 多足步行 |
| 危险等级 | ★★★★★（战场最高威胁） |

**外形描述**

要塞级的整体轮廓如同一头巨型等足目节肢动物（蜈蚣/球鼠妇），由多个膨大的圆球状体节串联而成，全高 66 米足以俯视多层建筑，与驱逐舰相比体量相当。体色以沙黄至乳白色为主，搭配深绛红色的步行肢，视觉上厚重而古老。

- **头部（前端体节）**：最前方体节呈圆润的穹顶状，表面有**四枚深色眼状感知器**，以对称方式排列于面部中央，表情漠然。头节两侧与顶部衔接着最前排的步行肢根部，颈部区域有蓝紫色关节甲板作为段间连接。
- **三胴体节结构**：身体由前胴、中胴、后胴三个主胴构成，每个主胴本身又由数个饱满膨胀的圆球状子节组成，子节之间以蓝紫色软质关节环相连。这些蓝紫色关节接合部即为已知的**唯一有效攻击弱点**。胴体表面呈乳黄/奶白色，布有细微裂纹状纹理，局部偶见橙红色色素斑。
- **步行肢（装甲足）**：共十余条，从各体节侧面伸出，极为粗长，色泽深绛红近黑，截面呈圆柱形，表面可见纵向纤维条束与环形节段，肢端为锐利锥形，踩踏力惊人。肢上部靠近躯干处带有蓝色或紫色瘤突状感知体。
- **尾节与冲角触手**：尾端体节向后延伸，末节细长，可向上蜷曲如蝎尾，平时收纳全长约 50 米的触手冲角。触手展开时如鞭，末端分叉为鲜红色的双弯钩状冲角，可分泌强力溶解液——战场图像显示其弧形展开时鲜红色冲角在高空划过，规模足以贯穿舰船结构。
- **小型种孵化机构**（官方设定）：步行肢根部附近存在特化的**生物孵化排出器官**。其运作方式：躯干肌肉隆起形成肥厚的肉节→肉节脱离并延伸出排出导管→以类"生物 3D 打印"方式从导管挤出半成型小型种 BETA→小型种在颤抖中立起后完成硬化、放出腺展开，最终成形离开母体投入战场。这一机制使要塞级在战场上具备**持续生产并部署小型种的能力**。

> 总体印象：古老而庞大的节肢巨兽，软体球节与坚硬血红长肢的对比极为鲜明；蓝紫色接合处是唯一的死穴，而其尾端那两枚鲜红弯钩与腹下持续"打印"涌出的小型种，共同构成了战场上最令人绝望的景象之一。

**特征**

- 已确认种类中**最大的多足步行大型种**（母舰级除外），防御力与耐久力极高
- 动作较为缓慢，对人类探测能力不高；**有效攻击点仅为三胴体构造的体节接合部**
- 主要攻击手段：
  - 用与要击级前臂同等硬度、韧性的**装甲脚踩踏**
  - 尾节收纳的全长约 **50 米带冲角触手**进行鞭击或刺击，前端**分泌强力溶解液**
- 体内可**收纳并释放大量小型种**（包括光线级），击破后务必确认安全

**战术建议**

> 攻击体节接合部弱点，避开装甲足踩踏范围。击破后**不可立即靠近**，需警惕体内光线级等小型种涌出。

## 三、光线属种

> ⚠️ 光线属种是影响人类制空权和远程火力支援的决定性威胁。  
> 其存在使得传统航空兵力近乎无法作战，是战场态势的根本改变者。

### 🔵 光线级（Laser-class）

| 项目 | 数值 |
|------|------|
| 全高 | 3 m |
| 全长 | 1.2 m |
| 全宽 | 1.6 m |
| 学名 | *Luxcius duo oculus* |
| 俗称 | Lux |
| 行动模式 | 二足步行 |
| 危险等级 | ★★★★★（制空封锁） |

**外形描述**

光线级拥有与其致命性极不相称的外貌——整体呈薄荷绿色，体表密布大量白色泡状肉疣，体型娇小，第一印象近乎"圆润可爱"，然而其学名 *Luxcius duo oculus*（双眼之光）与战场实绩揭示了其真正的恐怖。

- **头部（双穹顶瞄准器）**：头部占据全身比例极高，由两个巨大的穹顶状圆形结构并列组成，如同两枚硕大的眼球直接长于身体顶端。每个穹顶正面有一个巨大的深灰色圆形虹膜状构造，中央有白色高光反射点——这正是其**双眼激光瞄准/发射器官**。学名中的"duo oculus（双眼）"即指此结构。
- **躯干**：宽短而圆鼓，表面被密集的白绿色泡疣完全覆盖，泡疣大小不一、层叠堆砌，质感如同大型蜂巢或珊瑚礁，其内部疑为激光能量蓄积组织。躯干两侧伸出大量细长的针状或发丝状感觉毛束，绿色，向四周放射延伸，兼具感知环境与可能的散热功能。
- **下腹部开口**：躯干下部存在一道纵向裂口，平时紧闭，内部可见深红色的腔状组织——推测为**激光发射时的能量排出口或散热腔**，发射前会随能量充盈而撑开。
- **双足**：两条细长的绿白色步行腿从躯干底部伸出，末端小足带有暗红色血渍痕迹，支撑整个圆鼓身体，具备一定的机动移位能力。

> 总体印象：外形圆润、双眼巨大、色泽清新——光线级是已知 BETA 中外貌与威胁等级落差最大的种类。那双"大眼睛"不是在看你，而是在以精度锁定 380 公里外的飞行目标。

**特征**

- 最小的二足步行型光线属种，动作较为敏捷
- 可发射**足以破坏战术机与坦克**的高出力激光
- 激光照射间隔约 **12 秒**，可短时间内再次照射
- 最新对激光蒸发膜耐热防弹装甲被**最高出力照射约 5 秒即告失效**
- 防御力较低，能接近则以重机枪或战术机肢体打击即可给予充分杀伤

**战术建议**

> 优先清除。密度高时形成连续封锁。接近困难时，以炮兵或导弹从远距离摧毁。

### 🔵 重光线级（Heavy Laser-class）

| 项目 | 数值 |
|------|------|
| 全高 | 21 m（排热翼不计） |
| 全长 | 15 m |
| 全宽 | 11 m（排热翼不计） |
| 学名 | *Magnus Luxcius unioclus* |
| 俗称 | Magnus lux |
| 行动模式 | 二足步行 |
| 有效射程 | 100 km（高度 500 m 目标） |
| 再照射间隔 | 36 秒（最高出力时） |
| 危险等级 | ★★★★★（对重装备） |

**外形描述**

重光线级与光线级共享相似的体表构造，但体型更为庞大沉重，色调也从光线级的薄荷绿转变为**深棕红色**，整体散发出一种更为古拙的压迫感。学名 *Magnus Luxcius unioclus*（"单眼之大光"）精准点明其与光线级最显著的差异——由双眼变为**单眼**。

- **头部（单眼激光器）**：头顶前端生有一枚**极为巨大的单目圆球状感知/发射器官**，漆黑色，表面光滑如镜，带有一枚白色高光反射点。头部本体为锥形隆起，单眼就嵌于锥顶。这枚独眼即为**激光的瞄准与发射核心**，也是战术文件中提及的"照射黏膜"所在——可像眼睑一样展开保护皮膜将其覆盖，是其唯一已知弱点。
- **躯干**：极度圆鼓臃肿，由深棕红色基底组织与大量白色/灰白色泡状肉疣共同构成，疣体大而饱满，层叠覆盖躯干正面与侧面，其内部蓄积巨量激光发射所需能量。整体轮廓宽圆如一颗巨型肉球。
- **排热翼（侧翼散热结构）**：躯干侧面生有成簇的**深红色羽刺状排热翼**，由多根粗壮的绯红色棘刺放射展开，末端弯曲；此外另有一枚较大的弧形红色鳞甲翼片向后上方延展——这是重光线级为应对高出力激光射击产生的巨大废热而演化出的专用散热器官，未计入官方全高/全宽数据之中。
- **双足**：两条粗壮的淡色步行肢支撑庞大躯干，腿部明显比光线级更为粗重，行动迟缓。

> 总体印象：深棕圆体、独眼独目、侧翼血红——重光线级是光线属种的顶点形态，比光线级更加沉重、更加难以击杀；那枚能击沉战舰的独眼背后，是 36 秒再充能间隔这唯一的生机之窗。

**特征**

- 已知 BETA 中**最大的二足步行型光线属种**，动作缓慢但防御力高
- 照射出力**远高于光线级**，可在 100 公里外击落高度 500 米的低空飞行物
- 战舰耐热防弹装甲被持续照射 **10 秒即告蒸发**；推测射程本身可覆盖成层圈附近
- 唯一弱点——照射黏膜——可像眼睑一样展开保护皮膜加以防护
- 推荐以 **100 毫米以上炮弹**（战术机使用 120 毫米炮）攻击

**战术建议**

> 最高出力射击后有 **36 秒再充能窗口**，为唯一安全接近时机。需精确协同与极高机动性。

### 🔵 超重光线级（Super Heavy Laser-class）

| 项目 | 数值 |
|------|------|
| 全高 | ≈ 90 m |
| 首次确认 | 埃文斯克（樱花作战同期佯动战） |

| 防御力 | 极高 |
| 危险等级 | ★★★★★+（特级威胁） |

**外形描述**

超重光线级在尺寸与结构上完全突破了光线属种的固有形象，全高约 90 米，大幅超过要塞级（66 米），整体形如一座密布触手与炮管的活体要塞。体色以灰白至暗褐色为主，覆盖大量节状肉疣，局部嵌有紫黑色球形能量器官，发出幽幽蓝光。

- **三组超重照射头节（Scolex）**：从主体上方伸出**三条粗壮的柱状头节臂**，每条末端为多孔环形激光发射口（每组头节相当于三枚重光线级的激光器官，三组合计共九枚重光线级当量）。可同时齐射、也可交替快速连射（类加特林模式），最大出力可摧毁低轨道超音速飞行的装甲驱逐舰编队，射入太空时的等离子爆炸冲击波足以对地面装甲部队造成附带伤害。

- **超重能量囊（Energy Bag）**：主体中段嵌有**类反应堆的球形能量器官**，平时发出淡蓝色光芒。该器官可通过未知场效应对周围的光线级进行**非接触式无线能量供给**，使附近光线级的射击间隔显著缩短——这是超重光线级对战场态势影响最为深远的隐性能力之一。
- **超重近战囊（CQC Bag）**：前方两个腹段各收纳**50 条以上带冲角触手**（最大展开长度约 1,200 m）；背部有中型冲角触手（约 1,000 m）；腹面中段下方另有大型冲角触手（约 800 m），类似要塞级构造。埃文斯克战役中，该个体以此轻松击退数个中队规模的激光属种部队，并展现出令人震惊的近身格斗能力。
- **主体躯干**：宽大圆鼓，表面密布灰白色节状肉疣，与光线级、重光线级组织类型相似；嵌于疣间的紫黑色球形器官为能量储存/转化节点，发光时标志着大规模射击即将发动。
- **步行肢**：多条细长的深色尖刺状步行肢从躯干下方伸出，功能类似要塞级装甲足，但主要作为近战辅助与稳定支撑。

> 总体印象：超重光线级是已知 BETA 中兼具最强远程火力与强悍近战能力的种类，是真正意义上的战略级生物武器。三组九当量激光头节指向天空的那一刻，意味着任何高度的目标都不再安全。

**特征**

- 与樱花作战同期在埃文斯克出现的**超大型光线属种**，全高约 90 米
- **三组照射头节**，每组相当于 3 枚重光线级，合计出力可摧毁低轨道超音速编队
- 主体能量囊可无线供能给周围光线级，**显著缩短附近光线级的射击间隔**
- 近战触手最大展开长度约 **1,200 米**，埃文斯克实战中击退多个中队规模部队
- 三对超重排热翼散热量可生成**超大积雨云**，可干扰人类激光防御体系

**战术建议**

> 超重光线级为绝对最高威胁等级目标。远程激光封锁、超远程近战触手、无线供能周边光线种三重能力叠加，任何常规战术框架均告失效。**在其存在的战场上，制空权、轨道支援、远程炮兵全部失去意义**。唯一已知对策是战略级特殊作战。

- **整体轮廓**：极度细长的圆柱体，前端略为收束形成穿掘头部，中段为最大直径区域，向后端逐渐收细，尾端有较细的末节，末端生有橙褐色类爪状附属器。全长 1,800 米使其比任何已知地面大型种都更为庞大——要塞级（66 米）在其旁不过是一粒附属物。

- **前端口器（BETA 释放/穿掘器官）**：前端最显著结构——圆形大开口，内壁为**粉红色至洋红色平滑肉质组织**，开口内圈排列着**数圈大型橙黄色圆锥形齿状突起**，类似钻头的齿环，交错对称分布，直径足以让要塞级从中穿过。该开口兼具两项功能：穿掘作业时作为旋转钻头的核心破岩结构；抵达目标地点后开放，释放体内收纳的大量 BETA 部队。

**特征**

- 樱花作战中记录到的振动波形，在甲21号作战与横滨基地袭击中同样出现，推测**世界各地均有相当数量生息**

### ⚫ 头脑级（Brain-class）

| 项目 | 数值 |
|------|------|
| 全高/长/宽 | 不明 |
| 学名 | 巢穴反应炉（旧称） |
| 俗称 | 反应堆 |

| 危险等级 | ★★★★☆（战略价值极高） |

**外形描述**

头脑级是一种固定于巢穴核心大厅的**超大型球状发光生命体**，在人类发现之初被误认为是巢穴的"反应炉"装置——这一误判并不奇怪，因为它看上去更像一台有机能源机器，而非任何意义上的"生物"。

- **整体形态**：近似球形至卵形的巨大肉质团块，直径数十米，足以充满一座大型工业仓库的主体空间。周围常有人类修建的观测脚手架与起重机械，以供研究作业。

- **周围环境**：头脑级所在位置通常是巢穴最大的中央空腔，其散发的蓝光是整个巢穴内部照明的唯一来源。大量形体较小的 BETA 常聚集于其基部周围，接受能量补给。

> 总体印象：在黑暗的巢穴深处，头脑级如同一颗巨型发光的活体宝石，以无声而持续的蓝光滋养着整个巢穴中的每一个 BETA 个体。摧毁它，意味着整个巢穴的能量命脉被截断——但同时也意味着被捕人类的生命维持系统随之关闭。

**特征**

- 每个巢穴均拥有，被认为是核心存在
- 职能为**为 BETA 生物补给能量**，以及作为命令中枢接收并传达超重脑级的远距离指令
- **本身无任何思考能力**，完全由超重脑级操控，随后指挥 BETA 行动
- 樱花作战开始前，人类仅知其为「巢穴反应炉」，真实性质直至樱花作战方才确认

**战术意义**

> 摧毁头脑级可瘫痪整个巢穴的指挥与能量供给，但同时将切断对被捕人类的生命维持。为巢穴攻略的首要战略目标。

### ⚫ 超重脑级（Super Brain-class）

| 项目 | 数值 |
|------|------|
| 全高/长/宽 | 不明（超大型） |
| 学名 | 未定 |
| 俗称 | 无 |
| 位置 | 喀什 Original 巢穴最深处 |
| 数量 | **唯一** |
| 类型 | 固定型超大型头脑种 |
| 首次确认 | 樱花作战 |
| 危险等级 | ★★★★★+（全球战略级） |

**特征**

- 位于世界各地头脑级顶点的**唯一智慧性 BETA 种**，通过未知通信方式向全球发布指令
- 樱花作战时接通 ESP 发现体后**自称"上位存在"**——由此确认其智慧性
- 2001 年 12 月依据 ALTERNATIVE 4 调查研究，原先认定为「巢穴反应炉」的巨大设施被确认为头脑级，进而推算出超重脑级的存在
- 周围除门级外无其他 BETA，但**具有极高自我防卫能力**（事实上曾击毁 XG-70d 一架及武御雷一台）
- 依靠雌蕊状前端部位的**手状器官**进行战斗；触手状器官前端冲角强度推测与要击级前肢及突击级装甲壳相当
- 确认具备**使 Rutherford 场失效**的能力，以及**控制所接触的计算机与人类头脑**的能力

**战术意义**

> 超重脑级是人类反攻的终极目标（即「樱花作战」核心目的）。摧毁超重脑级理论上可终止 BETA 的协调性入侵行为。任何常规手段均无法直接处置，需战略级特殊作战方可达成。
